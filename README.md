# 📊 台股 AI 戰情室 Pro Max

整合技術面、籌碼面、AI 智能分析的專業台股看盤系統

---

## 🎯 系統特色

### 📈 技術分析功能
- **K 線圖表**：紅漲綠跌的專業 K 線呈現
- **多週期均線系統**：MA5/10/20/60 完整均線
- **MACD 指標**：DIF、MACD、柱狀圖三合一
- **KD 隨機指標**：含超買超賣警示線
- **RSI 相對強弱**：14 日 RSI 指標
- **布林通道**：上軌、下軌、通道寬度

### 💼 籌碼分析功能
- **外資買賣超**：追蹤外資動向
- **投信操作**：掌握法人作帳
- **自營商部位**：了解券商策略
- **主力合計**：外資+投信總和
- **融資餘額**：散戶情緒指標
- **融券餘額**：軋空風險評估

### 🤖 AI 智能分析
- **技術面診斷**：趨勢、指標、支撐壓力
- **籌碼面解密**：法人動向、散戶籌碼
- **綜合研判**：操作建議、風險提示
- **Powered by**: Google Gemini 2.0 Flash

## 🚀 快速開始

### 步驟 1：安裝 Python
1. 前往 [Python 官網](https://www.python.org/downloads/) 下載
2. 安裝時勾選「Add Python to PATH」
3. 完成安裝

### 步驟 2：解壓縮系統檔案
將整個資料夾解壓縮到任意位置

### 步驟 3：啟動系統

**Windows 使用者：**
雙擊 `啟動系統.bat`

**Mac/Linux 使用者：**
```bash
./啟動系統.sh
```

首次啟動會自動安裝所需套件（需要幾分鐘）

### 步驟 4：使用系統
1. 瀏覽器會自動開啟 http://localhost:8501
2. 在左側輸入股票代碼（例如：2330）
3. 點擊「開始分析」

## 🐛 常見問題

### Q1：執行時出現「找不到模組」錯誤
**A**：刪除 `installed.flag` 檔案，重新啟動系統讓它重裝套件

### Q2：圖表無法顯示
**A**：
1. 檢查網路連線
2. 確認股票代碼正確（台股代碼，例如 2330）
3. 嘗試其他股票代碼

### Q3：AI 分析顯示錯誤
**A**：
1. 確認 API Key 正確
2. 檢查 API 使用額度是否用盡
3. 稍後再試（可能是 API 暫時繁忙）

### Q4：某些欄位顯示為 0
**A**：
- 法人或融資數據可能當日無交易
- 週末假日無新數據
- FinMind API 可能暫時無法取得

### Q5：系統速度很慢
**A**：
1. 減少分析天數（60-90 天較快）
2. 關閉其他耗資源程式
3. 資料會快取 1 小時，重複查詢會較快

如果自動安裝失敗，可手動執行：

```bash
pip install streamlit yfinance pandas plotly google-generativeai FinMind
```

### 更新系統
```bash
pip install --upgrade -r requirements.txt
```

### 清除快取
刪除以下資料夾：
- Windows: `C:\Users\您的使用者名稱\.streamlit`
- Mac/Linux: `~/.streamlit`
